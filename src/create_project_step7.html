Use this page to define all variables that will be balanced among treatment groups using the minimization algorithm. Each variable name and variable level should consist of standard alphanumeric symbols, and should not contain spaces or commas. The levels should be entered as a comma separated list of labels for the different levels of a variable. For example, if the variable name is "Gender", the levels could be entered as "Female,Male". The "Weight" parameter (a positive number) determines how much influence each variable has in determining the group assignments (variables with greater weight have greater influence).

The "function" parameter determines what function the minimization algorithm optimizes when assigning subjects to treatment groups. The default "range" option aims to minimize the ranges (largest minus smallest value) among the fractions of subjects within each level of a variable that are assigned to the different treatment groups. These ranges are summed across all the levels of all the variables to produce a final score to minimize. For example, if there are three treatment groups, and the proportions of females assigned to the three groups are 0.3, 0.6, 0.5, then the range is 0.6 - 0.3 = 0.3. The "standard deviation" option for the minimization algorithm aims to minimize the standard deviation among these fractions, summed over all levels of all variables, i.e. the standard deviation of 0.3, 0.5, 0.6 would be one term in this sum. In most cases, the default "range" function is the better choice.
